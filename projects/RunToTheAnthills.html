Run to the Anthills

Project link | Codebase | Unity, C# | September 2021

This game was made for the one-week long GBJAM 9, which theme was Original GameBoy, using Unity 2020.3.

The gameplay is entirely based on the Langton's ant: the player watches an ant that moves following a specific pattern. However, it is possible to modify the tiles and thus to have control over the path the ant follows. On top of that, three other ants can be unlocked with each one having their own pattern.
I made sure to respect the specific limitations the Game Boy had to face such as sprite limits, color palettes, controls, audio channels and more.
