Enigma

Project | Codebase | Unity, C# | July 2019

This game was made for the Weekly Game Jam 105, which theme was Cheat To Win, using Unity 2019.2.

It is a very simple game taking place in a dark room, containing ten buttons on the wall and a single NPC: Minako, a mysterious talking door. Its name is an anagram of Konami, giving a hint on how to resolve the puzzle. The only way to escape the room is indeed to form the Konami Code by interacting with the buttons on the wall.
It was my very first game jam and although it was a week-long, I tried to keep the game as simple as possible, while focusing on the main features such as the dialog system, or using lights efficiently to ensure the mysterious atmosphere of the room.
