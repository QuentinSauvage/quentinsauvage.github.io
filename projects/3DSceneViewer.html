3D Scene Viewer

Project and codebase | C++, SDL | December 2019

This program was made as a school project during my last year of Master and allows to view, move and rotate 3D geometric objects in a simplified way compared to classic game engines. It takes as an input a list of files defining the position of each vertex for each object, in order to draw the triangles of the meshes.

It is written in C++ and uses the SDL library. I was asked to recreate every vector, matrix and quaternion operations (dot and cross product, transposition, normalisation, etc.) as well as functions related to the transform property of the objects. It contains the definition for some geometric primitives (sphere, plane, triangle, etc.) and finally operations related to the camera.
