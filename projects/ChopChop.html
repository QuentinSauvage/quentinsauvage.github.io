Chop-Chop!

Project link

This game was made for the GE Jam #3, which theme was Duality, using Unity 2020.2. In the end, we loved working on this project and decided to polish and submit it to the Finally Finish Something 2021 jam.

It is an action game heavily inspired by the great Kuri Kuri Mix (or The Adventures of Cookie & Cream) made by From Software in 2000. The concept is to control two characters with only one controller, using for instance the left joystick (resp., the right joystick) to move the character on the left (resp. the character on the right). There is a time limit and the two characters need to cooperate in order to reach the end in time.
I worked on the controls (keyboard, controller) and character aspects of the game as long as the UI, gameplay elements (spikes, pits, etc.). I also integrated the animations, art and audio for which we used free assets. It has been the perfect way to familiarize with the new Input System of Unity that significantly changed since its beta (which I mildly used previously). I also created some simple shaders using Shader Graph, in order to familiarize myself with the new workflow offered by the 2020.2 version.
