Accelerando

Project link | Godot, GDScript | February 2024

This game was made for the one-week long NOKIA 3310 Jam 6, which theme was Freeze, using Godot 4.2.

It is one of my first meaningful project using Godot and I had to pay attention to the Nokia 3310 limitations, so I kept things simple. The game follows a BPM that increases over time and periodically changes the musical key selected. Notes freeze over time and they freeze even faster if they are part of the current key. To unfreeze a note, the player must press the key associated to that note on their keyboard. When it happens, the note is played and its state is reset. Once a note is completely frozen, the game ends and plays a music based on the player's inputs during the game.
