C.A.T.S. Attacks

Project | Codebase | Godot, GDScript | December 2024

This game was made for the Secret Santa Jam 2024, using Godot 4.4.

Our giftee asked for a bullet hell game with some exploration so that's what we did. The game alternates between 3D exploration where the player interacts with the environment and bullet hell levels with firing patterns that follow the music. Synchronizing the bullet patterns with the musics ended up being way harder than we expected so we had to sacrifice the visual quality of the game.
