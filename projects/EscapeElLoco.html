El Loco Racing

Project link | October 2019

This game was made as a school project during my last year of Master, using C++ and SFML. Its purpose was to familiarise myself with collision handling and the game loop management in order to maintain a correct framerate.

It is a small game where the player controls two characters at the same time, with the same controls: if one character goes left, the other one will go left too. The goal is to cross a door with each character, but before that, they have to unlock these doors by collecting a key hidden in the level.
Each character progresses on a separate part of the level and can collect coins which will temporarily make blocks appear on the other side, helping the respective player to reach the end of the level.
