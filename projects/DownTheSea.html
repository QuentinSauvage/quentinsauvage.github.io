Down the Sea

Project link | March 2021

This game was made for the one-week longMundane Jam (8 Bits to Infinity), using Unity 2020.2.

It is a simple gestion game where the player dispatches its fishing boats, makes money at the end of each day which then allows to buy better equipments and boats. Special events happen randomly in order to raise the difficulty and make each game a bit unique. There's also a basic fishing minigame that allows the player to make more money.
