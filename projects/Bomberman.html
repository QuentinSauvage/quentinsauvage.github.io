Bomberman

Project link | October 2018 - December 2018

This game was made as a school project during my first year of Master, for the "Networking" school module. It is an online version of Bomberman with the server-side written in C and the client in Java, using the MVC design pattern.
In addition, it contains game bonuses such as an increased bomb power, increased health or the ability to slow down opponents.
