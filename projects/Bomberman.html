Bomberman

Project link | October 2018 - December 2018

This game was made as a school project during my first year of Master, for the "Networking" school module. It is an online and classic Bomberman with a C-written server side and the client part in Java with the MVC design pattern.
In addition, it contains game bonus such as an increased bomb power, increased health or a slow effect.
