Pac-Man

Project link

This school project has been made at the end of my year of studies (2015) and is written in C++, using SFML. It has the most important features of the classic Pac-Man, such as game mechanics, specific AI for each ghost and a score system. On top of that, there are also several new features implemented in the game: a hard mode (which makes the screen spin), a Lord of the Rings theme and a very basic account system.
