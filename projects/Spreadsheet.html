Spreadsheet Shell

Project link | December 2017

This project was made as a school project during my third year of Bachelor. It takes as input a spreadsheet with mathematical operations between cells as Excel does (e.g. =C1+B2), and gives the original spreadhsheet with every operations computed. Inputs and outputs can either be directly in the CLI, or in a text file.

The programm has been fully written in Bash. It contains some basic function keywords for operations, works with primitive types (boolean, integer, float and strings). Variables can also be used in the spreadsheets given to the program.
