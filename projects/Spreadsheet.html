Spreadsheet Shell

Project and codebase | Bash | December 2017

This bash project was made as a school project during my third year of Bachelor. It takes as an input a spreadsheet with mathematical operations between cells just like Excel (e.g. =C1+B2) and outputs the original spreadhsheet with every operations computed. Inputs and outputs can either be given directly inside the CLI, or using a text file.

The programm has been fully written in Bash. It contains some basic function keywords for operations and works with primitive types. Variables can also be used in the spreadsheets fed to the program.
