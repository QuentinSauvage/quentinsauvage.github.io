Equilibrium

Project link | July 2020

This game was made for the GMTK Game Jam 2020, which theme was Out of Control, using Unity 2019.4.

It is a labyrinth game in which the player must control properties such as hunger, thirst or happiness to stay alive until it reaches the end and some properties interact between them. For instance, if the player is hungry, its speed will be affected and it will also become angry.
I worked on the full logic of the game by creating a completely modular system using Scriptable Objects allowing to control every interaction with the game data. I discovered weeks later Atom, which shares the same objectives as my module but is also much more complete. On top of that, as it was a two-day-long jam, I only had the time to work on the game's audio and the level design.
