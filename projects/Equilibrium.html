Equilibrium

Project link | Unity, C# | July 2020

This game was made for the GMTK Game Jam 2020, which theme was Out of Control, using Unity 2019.4.

It is a labyrinth game in which the player must control its attributes such as hunger, thirst or happiness to stay alive until it reaches the end. On top of that, an attribute can act on other attributes. For instance, if the player is hungry, its speed will be affected and it will also become angry dealing more damage.
I worked on the full logic of the game by creating a completely modular system using Scriptable Objects allowing to control every interaction based on the game data. I discovered some weeks later Atom, which shares the same objectives as my module but is also much more complete. On top of that, as it was a two-day-long jam, I barely had time to work on the game's audio and level design.
