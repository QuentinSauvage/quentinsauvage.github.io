Le Rembobineur

Project link | August 2020

This game was made for the Brackeys Jam 2020.2, which theme was Rewind, using Unity 2020.1.

It is a puzzle game where the player has to reach the end of the level without getting caught by the guards. As the character automatically follow a path to the end of the level, the player needs to interact with elements of the game in order to clear its way. Some levels end on a boss fight where the player has to dodge obstacles.
I mostly worked on all the UI interactions, game and level design, animations, art assets research, their integration and the game logic. I also decided to compose every music of the game.
