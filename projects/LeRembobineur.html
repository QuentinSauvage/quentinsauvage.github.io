Le Rembobineur

Project | Codebase | Unity, C# | August 2020

This game was made for the Brackeys Jam 2020.2, which theme was Rewind, using Unity 2020.1.

It is a puzzle game where the player has to reach the end of the level without getting caught by the guards. As the character automatically follows a path to the end of the level, the player needs to interact with elements of the game in order to clear the way. Some levels end on a boss fight where the player simply has to dodge obstacles.
I mostly worked on all the UI interactions, game and level design, animations, assets research, their integration and the game logic. I also worked on the music of the game.
