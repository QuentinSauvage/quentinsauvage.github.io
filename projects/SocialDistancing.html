Social Distancing

Project | Codebase | Unity, C# | April 2020

This game was made for the Easy Jam (8 bits to Infinity), which theme was No Enemies, using Unity 2019.4.

It is a life simulator inspired by games like Stardew Valley in which the player can fish by using baits and grow vegetables in order to make money.
I worked on the 3C, vegetables growth and the time and weather change mechanic. I deepened my experience with serialized objects thanks to this jam as I used them to represent various data for the game such as vegetables and fish.
