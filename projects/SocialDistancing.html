Social Distancing

Project link April 2020

This game was made for the Easy Jam (8 bits to Infinity), which theme was No Enemies, using Unity 2019.4.

It is a life simulator inspired by games like Stardew Valley in which the player can fish by using baits and grow vegetables in order to make money.
I worked on the 3C, vegetables growth and the time and weather change mechanic. Thanks to this jam, I had my first real experience with serialized objects that I used to represent vegetables and fish.
